This repo includes most used drivers for avr atmega 32

LIB =>
      1- Types
      2- Bit Math
MCAL =>
      1- DIO
      2- ADC
      3- GI
      4- EXTI
      5- TIMER0
      6- TIMER1
      7- TIMER2
      8- ICU
      9- WDT
      10- UART
      11- SPI
      12- TWI (I2C)
HAL=>
      1- 7 SEGMENT
      2- Key Pad
      3- LCD
      4- Stepper Motor
      5- Servo Motor
      6- External EEPROM
